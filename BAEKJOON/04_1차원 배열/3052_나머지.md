## 💡 문제 이해
> 문제를 나만의 언어로 요약해서 적어보세요.

- **문제**: 
  두 자연수 A와 B가 있을 때, A%B는 A를 B로 나눈 나머지 이다. 예를 들어, 7, 14, 27, 38을 3으로 나눈 나머지는 1, 2, 0, 2이다.
  
  수 10개를 입력받은 뒤, 이를 42로 나눈 나머지를 구한다. 그 다음 서로 다른 값이 몇 개 있는지 출력하는 프로그램을 작성하시오.

- **핵심 조건:** 첫째 줄에, 42로 나누었을 때, 서로 다른 나머지가 몇 개 있는지 출력한다.

## 💻 정답 코드 (Java)

```java
import java.io.BufferedReader;
import java.io.BufferedWriter;
import java.io.IOException;
import java.io.InputStreamReader;
import java.io.OutputStreamWriter;

public class Main {

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        BufferedWriter bw;
        bw = new BufferedWriter(new OutputStreamWriter(System.out));
		
        boolean[] check = new boolean[42];
		
        for (int i = 0; i < 10; i++) {
            int num = Integer.parseInt(br.readLine());
			
            check[num % 42] = true;
        }
		
	    int totalCount = 0;
        for (boolean b : check) {
            if (b) {
                totalCount++;
            }
        }
		
        bw.write(String.valueOf(totalCount));
        bw.flush();
        bw.close();
        br.close();
    }
}
```

## 📝 오답 노트 & 회고
> 틀렸다면 왜 틀렸는지, 맞았다면 더 좋은 방법은 없는지 기록하세요.

- **틀린 이유:** 
	- **이중 반복문 로직 오류:** 두 값을 비교하려고 했으나, 인덱스 변수(`n`, `a`)가 같이 증가하도록 잘못 작성되어 자기 자신끼리만 빼는 연산을 수행함. (결과가 무조건 100이 나옴)
	- **`BufferedWriter` 사용 미숙:** `bw.write(int)`는 숫자를 그대로 출력하는 것이 아니라, 해당 **아스키 코드에 매핑된 문자**를 출력한다는 점을 간과함.

- **배운 점:** 
	- **인덱스를 활용한 카운팅 (Frequency Array):** 서로 다른 숫자의 개수를 셀 때, 복잡하게 비교하기보다 **값 자체를 배열의 인덱스(0~41)**로 사용하여 `boolean` 배열에 체크하는 방식이 훨씬 효율적이고 구현이 쉽다.
    
	- **출력 형변환:** `BufferedWriter`로 숫자를 출력할 때는 반드시 `String.valueOf()`를 사용해 문자열로 변환해야 한다.

- **다른 사람 풀이 참고:** **`HashSet` 활용:** 자바의 `Set` 자료구조는 **중복을 허용하지 않는 특성**이 있어, 나머지를 `add`만 하고 마지막에 `size()`를 출력하면 코드가 5줄 이내로 끝난다.
