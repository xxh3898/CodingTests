# 2025-11-22

## 💡 문제 이해
> 문제를 나만의 언어로 요약해서 적어보세요.

- **문제**: 
	N을 입력받은 뒤, 구구단 N단을 출력하는 프로그램을 작성하시오. 출력 형식에 맞춰서 출력하면 된다.

- **핵심 조건:** 입력 받은 수를 1~9까지 곱해서 출력

---

## 🚀 접근 방식 (Logic)
> 코드를 짜기 전, 어떤 알고리즘과 자료구조를 쓸지 설계하는 단계입니다.

1. **변수 설정:** 입력받을 `n`을 정수로 선언한다. 

2. **조건문 설계:** 
	- 1~9까지 반복하여 곱한다.

---

## 💻 정답 코드 (Java)

> 백준허브가 자동 생성한 문제 설명과 풀이 코드를 직접 엽니다.

- [[CodingTests/백준/Bronze/2739. 구구단/README|문제 설명 보기]]
- [[CodingTests/백준/Bronze/2739. 구구단/구구단.java|풀이 코드 보기]]

```java
import java.util.Scanner;

public class Main {

    public static void main(String[] args) throws Exception {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        for (int i = 1; i <= 9; i++) {
            System.out.printf("%d * %d = %d\n", n, i, (n * i));
        }
    }
}
```
