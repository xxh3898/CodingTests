## 💡 문제 이해
> 문제를 나만의 언어로 요약해서 적어보세요.

- **문제**: n이 주어졌을 때, 1부터 n까지 합을 구하는 프로그램을 작성하시오.

- **핵심 조건:** n까지 반복하여 더함

---

1. **변수 설정:** 입력받을 `n`을 정수로 선언한다. 

2. **조건문 설계:** 
	- while(i  <  n)

## 💻 정답 코드 (Java)

> 백준허브가 자동 생성한 문제 설명과 풀이 코드를 직접 엽니다.

- [[CodingTests/백준/Bronze/8393. 합/README|문제 설명 보기]]
- [[CodingTests/백준/Bronze/8393. 합/합.java|풀이 코드 보기]]

```java
import java.util.Scanner;

public class Main {

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        int i = 0, sum = 0;
        while (i < n) {
            i++;
            sum = i + sum;
        }
        System.out.println(sum);
    }
}
```
