# 2025-11-28

## 💡 문제 이해
> 문제를 나만의 언어로 요약해서 적어보세요.

- **문제**: 총 N개의 정수가 주어졌을 때, 정수 v가 몇 개인지 구하는 프로그램을 작성하시오.

- **핵심 조건:** 입력이 없으면 종료

---

## 🚀 접근 방식 (Logic)
> 코드를 짜기 전, 어떤 알고리즘과 자료구조를 쓸지 설계하는 단계입니다.

1. **변수 설정:** 입력받을 `n`을 정수로 선언한다. 

2. **조건문 설계:** `if (num == arr[i]) {count = count + 1;}`

## 💻 정답 코드 (Java)

- [[CodingTests/백준/Bronze/10807. 개수 세기/개수 세기.java|풀이 코드 보기]]

```java
import java.io.BufferedReader;
import java.io.BufferedWriter;
import java.io.IOException;
import java.io.InputStreamReader;
import java.io.OutputStreamWriter;
import java.util.StringTokenizer;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));
        int n = Integer.parseInt(br.readLine());
        int count = 0;
        StringTokenizer st = new StringTokenizer(br.readLine());
		
        int[] arr = new int[n];
        for (int i = 0; i < n; i++) {
            arr[i] = Integer.parseInt(st.nextToken());
        }
        int num = Integer.parseInt(br.readLine());
		
        for (int i = 0; i < n; i++) {
            if (num == arr[i]) {
                count = count + 1;
            }
        }
        bw.write(String.valueOf(count));
        bw.flush();
        bw.close();
    }
}
```

- **배운 점:** 숫자를 문자열(String)로 바꿔서 넣어주어야 한다.
