# 2025-12-09

## 💡 문제 이해
> 문제를 나만의 언어로 요약해서 적어보세요.

- **문제**: 문자열을 입력으로 주면 문자열의 첫 글자와 마지막 글자를 출력하는 프로그램을 작성하시오.

- **핵심 조건:** 문자열의 길이가 1일 땐 문자 두 번 출력

---

## 🚀 접근 방식 (Logic)
> 코드를 짜기 전, 어떤 알고리즘과 자료구조를 쓸지 설계하는 단계입니다.

1. **변수 설정:** 입력받을 `n`을 정수로 `s`를 문자열로 선언한다.

2. **조건문 설계:** `s`의 길이가 1일 경우 `s`를 두 번 출력

3. **시간 복잡도:** O(1)

## 💻 정답 코드 (Java)

> 백준허브가 자동 생성한 문제 설명과 풀이 코드를 직접 엽니다.

- [[CodingTests/백준/Bronze/9086. 문자열/README|문제 설명 보기]]
- [[CodingTests/백준/Bronze/9086. 문자열/문자열.java|풀이 코드 보기]]

```java
import java.io.BufferedReader;
import java.io.BufferedWriter;
import java.io.IOException;
import java.io.InputStreamReader;
import java.io.OutputStreamWriter;
import java.util.StringTokenizer;

public class Main {
	
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));
        StringTokenizer st = new StringTokenizer(br.readLine());
        int n = Integer.parseInt(st.nextToken());
        String s = "";
		
        for (int i = 0; i < n; i++) {
            s = br.readLine();
            int l = s.length();
			
            if (s.length() == 1) {
                bw.write("" + s + s + "\n");
            } else {
                bw.write("" + s.charAt(0) + s.charAt(l - 1) + "\n");
            }
        }
        bw.close();
        br.close();
    }
}
```

## 📝 오답 노트 & 회고
> 틀렸다면 왜 틀렸는지, 맞았다면 더 좋은 방법은 없는지 기록하세요.

- **틀린 이유:**
  자바는 연산을 왼쪽에서 오른쪽으로 수행합니다.
	1. `s.charAt(0)`과 `s.charAt(l - 1)`은 둘 다 **문자(char)**입니다.
	2. 문자끼리 `+` 연산자가 만나면 자바는 이를 **아스키 코드(ASCII) 값의 덧셈**으로 처리합니다.
	    - 예를 들어 입력이 "AB"라면:
	    - 'A' (65) + 'B' (66) = **131** (정수)
	3. 그 뒤에 `+ "\n"`을 만나면서 `131`이라는 숫자가 문자열로 변환되어 줄바꿈과 합쳐집니다.
	4. 결과적으로 `AB`가 아니라 `131`이 출력됩니다.

- **배운 점:** 가장 앞에 **빈 문자열(`""`)** 을 더해주면 된다. 문자열이 연산의 가장 앞에 오면, 그 뒤에 오는 `char`들도 자동으로 문자열로 변환되어 연결됨
