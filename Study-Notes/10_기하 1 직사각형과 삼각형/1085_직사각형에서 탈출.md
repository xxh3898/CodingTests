## 💡 문제 이해
> 문제를 나만의 언어로 요약해서 적어보세요.

- **요약**: 현재 위치 $(x, y)$에서 직사각형의 경계선(왼쪽, 오른쪽, 위, 아래)까지 가는 거리 중 **가장 짧은 거리(최솟값)**를 구하는 문제.
- **핵심 조건**: 
	- 직사각형은 $(0, 0)$부터 $(w, h)$까지이다.
	- 비교해야 할 거리는 총 4개이다.
	    1. 왼쪽 경계선까지: $x$
	    2. 오른쪽 경계선까지: $w - x$
	    3. 아래쪽 경계선까지: $y$
	    4. 위쪽 경계선까지: $h - y$

---

## 🚀 접근 방식 (Logic)
> 코드를 짜기 전, 어떤 알고리즘과 자료구조를 쓸지 설계하는 단계입니다.

1. **변수 설정:** 입력받은 `x`, `y`, `w`, `h`.
2. **최솟값 찾기:** 
	- 가로 기준 최단 거리: $x$와 $w-x$ 중 작은 값.
	- 세로 기준 최단 거리: $y$와 $h-y$ 중 작은 값.
	- 최종적으로 위 두 값 중 더 작은 값을 출력한다.
3. **자료구조/함수:** `Math.min()` 함수를 사용하면 간결하게 구현 가능.

## 💻 소스 코드 및 문제 설명
> 백준허브가 자동 생성한 **문제 폴더**로 이동합니다.

- [📂 문제 풀이 폴더 열기 (소스코드 & README)](../../백준/Bronze/1085. 직사각형에서 탈출/)

```java
import java.io.BufferedReader;
import java.io.BufferedWriter;
import java.io.IOException;
import java.io.InputStreamReader;
import java.io.OutputStreamWriter;
import java.util.StringTokenizer;

public class Main {

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));
        StringTokenizer st = new StringTokenizer(br.readLine());

        int x = Integer.parseInt(st.nextToken());
        int y = Integer.parseInt(st.nextToken());
        int w = Integer.parseInt(st.nextToken());
        int h = Integer.parseInt(st.nextToken());

        // 1. 가로 기준 최소 거리 (왼쪽 벽 vs 오른쪽 벽)
        int minX = Math.min(x, w - x);
        
        // 2. 세로 기준 최소 거리 (아래 벽 vs 위 벽)
        int minY = Math.min(y, h - y);

        // 3. 둘 중 더 작은 값이 전체 최소 거리
        bw.write(Math.min(minX, minY) + "");

        bw.close();
        br.close();
    }
}
```

## 📝 오답 노트 & 회고
> 틀렸다면 왜 틀렸는지, 맞았다면 더 좋은 방법은 없는지 기록하세요.

- **틀린 이유:** 
	1. **`else if`의 함정**: `if`문에서 `min > x`가 참이 되어버리면, 그 뒤에 있는 `else if` 문들은 아예 실행되지 않고 건너뛴다.
	     - 예: $x=10, y=5$일 때, $10$이 초기값($1000$)보다 작아서 `min`이 10이 되고 끝난다. 실제로는 $y(5)$가 더 작은데 비교조차 하지 못함. -> **모든 `if`를 개별적으로 쓰거나 `Math.min`을 써야 함.**
	  2. **불필요한 `+1`**: 문제는 단순 '거리'를 묻는 것이므로 좌표 차이를 그대로 출력해야 한다.
- **배운 점:** 최솟값을 구할 때는 조건문의 흐름(Flow)에 주의해야 하며, 자바의 `Math.min(a, b)`을 활용하면 가독성과 정확도를 높일 수 있다.````
