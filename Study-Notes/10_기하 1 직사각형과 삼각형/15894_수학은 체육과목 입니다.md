## 💡 문제 이해
> 문제를 나만의 언어로 요약해서 적어보세요.

- **요약**: 가장 아랫부분의 정사각형 개수 $N$이 주어졌을 때, 피라미드 모양 도형의 **실선(둘레)의 길이**를 구하는 문제.
- **핵심 조건**: 
	- $1 \le N \le 10^9$ ($10$억).
	- 결과값은 $N$에 비례하여 매우 커질 수 있다.
	- 시간 제한은 1초이다.

---

## 🚀 접근 방식 (Logic)
> 코드를 짜기 전, 어떤 알고리즘과 자료구조를 쓸지 설계하는 단계입니다.

1. **규칙 찾기 (둘레 공식 유도):** 
	- 윗변(가로): $N$개
	- 아랫변(가로): $N$개
	- 높이(세로): 왼쪽 계단들의 합은 전체 높이 $N$과 같고, 오른쪽 계단들의 합도 $N$과 같다.
	- **총 둘레**: $N(\text{위}) + N(\text{아래}) + N(\text{좌}) + N(\text{우}) = 4 \times N$
	
2. **자료형 선택 (중요!):** 
	- $N$의 최대값은 $10^9$이다.
	- 답은 $4 \times N$이므로 최대 $4 \times 10^9$ ($40$억)이 된다.
	- Java의 `int`형 최대 범위는 약 $21$억($2 \times 10^9$)이다.
	- 따라서 $40$억을 담기 위해서는 반드시 **`long`** 타입을 사용해야 한다.

## 💻 소스 코드 및 문제 설명
> 백준허브가 자동 생성한 **문제 폴더**로 이동합니다.

- [📂 문제 풀이 폴더 열기 (소스코드 & README)](../../백준/Bronze/15894. 수학은 체육과목 입니다/)

```java
import java.io.BufferedReader;
import java.io.BufferedWriter;
import java.io.IOException;
import java.io.InputStreamReader;
import java.io.OutputStreamWriter;

public class Main {

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));

        // 입력값이 10억까지 들어오므로, 연산 결과를 위해 long으로 받거나 형변환이 필요함
        long n = Long.parseLong(br.readLine());

        // 둘레는 항상 n * 4
        bw.write((n * 4) + "");

        bw.close();
        br.close();
    }
}
```

## 📝 오답 노트 & 회고
> 틀렸다면 왜 틀렸는지, 맞았다면 더 좋은 방법은 없는지 기록하세요.

- **틀린 이유:** - 작성한 수식 `1 + n * 3 + (n - 1)`은 정리하면 `4n`이 되어 논리는 맞았음.
	- 하지만 $N$이 $10^9$일 때, $4N$은 $40$억이 되어 `int` 범위를 넘어섬(**Overflow 발생**).
	- 이로 인해 음수나 엉뚱한 값이 출력되어 틀림.
- **배운 점:** 
	- 입력값의 범위($10^9$)를 보고 결과값이 `int` 범위를 넘을지 미리 계산해보는 습관이 필요함.
	- $21$억이 넘어갈 것 같으면 무조건 `long`을 사용해야 함.
