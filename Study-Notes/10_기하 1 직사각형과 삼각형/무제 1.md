## 💡 문제 이해
> 문제를 나만의 언어로 요약해서 적어보세요.

- **요약**: 세 변의 길이가 주어졌을 때, 삼각형의 종류(Equilateral, Isosceles, Scalene)를 판별하거나 삼각형이 성립하지 않는 경우(Invalid)를 출력하는 문제.
- **핵심 조건**: 
	- **종료 조건**: 입력으로 `0 0 0`이 들어오면 프로그램을 종료한다.
	- **삼각형 성립 조건 (Invalid 체크)**: `가장 긴 변 >= 나머지 두 변의 합`이면 삼각형이 될 수 없다.
	- **순서 주의**: 성립 조건을 만족하지 못하면 다른 조건(이등변 등)보다 우선하여 `Invalid`를 출력해야 한다. (단, 정삼각형은 예외가 없으므로 순서가 유연함)

---

## 🚀 접근 방식 (Logic)
> 코드를 짜기 전, 어떤 알고리즘과 자료구조를 쓸지 설계하는 단계입니다.

1. **무한 루프 설계:** `while(true)`를 사용하여 계속 입력을 받고, `0 0 0`이 들어오면 `return` 또는 `break`로 종료한다.
2. **핵심 로직 (나머지 두 변의 합 구하기):**
	- 굳이 중간값을 찾으려 애쓸 필요가 없다.
	- `나머지 두 변의 합` = `(세 변의 총합) - (가장 긴 변)`
	- 이를 이용해 `Invalid` 조건인 `max >= (sum - max)`를 쉽게 구현한다.
3. **조건문 분기:**
	- **Step 1:** 세 변이 모두 같으면 `Equilateral`.
	- **Step 2:** 삼각형 성립 조건을 어기면 `Invalid`.
	- **Step 3:** 세 변이 모두 다르면 `Scalene`.
	- **Step 4:** 그 외의 경우(두 변만 같음)는 `Isosceles`

## 💻 소스 코드 및 문제 설명
> 백준허브가 자동 생성한 **문제 폴더**로 이동합니다.

- [📂 문제 풀이 폴더 열기 (소스코드 & README)](../../백준/Bronze/5073. 삼각형과 세 변/)

```java
import java.io.BufferedReader;
import java.io.BufferedWriter;
import java.io.IOException;
import java.io.InputStreamReader;
import java.io.OutputStreamWriter;
import java.util.StringTokenizer;

public class Main {

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));
        StringTokenizer st;

        while (true) {
            String reesult = ""; // 결과 문자열
            st = new StringTokenizer(br.readLine());
            
            int a = Integer.parseInt(st.nextToken());
            int b = Integer.parseInt(st.nextToken());
            int c = Integer.parseInt(st.nextToken());

            // 1. 종료 조건
            if (a == 0 && b == 0 && c == 0) {
                bw.close();
                br.close();
                return;
            }

            // 2. 판별 로직
            // 세 변이 모두 같은 경우 (정삼각형)
            if (a == b && b == c && c == a) {
                reesult = "Equilateral";
                bw.write(reesult + "\n");
            } 
            // 삼각형 성립 조건 위배 (가장 긴 변 >= 나머지 합)
            // 논리: (총합 - 최댓값)은 나머지 두 변의 합과 같다.
            else if (Math.max(a, Math.max(c, b)) >= (a + b + c) - Math.max(a, Math.max(c, b))) {
                reesult = "Invalid";
                bw.write(reesult + "\n");
            } 
            // 세 변이 모두 다른 경우
            else if (a != b && b != c && c != a) {
                reesult = "Scalene";
                bw.write(reesult + "\n");
            } 
            // 두 변이 같은 경우 (이등변)
            else {
                reesult = "Isosceles";
                bw.write(reesult + "\n");
            }
        }
    }
}
```

## 📝 오답 노트 & 회고
> 틀렸다면 왜 틀렸는지, 맞았다면 더 좋은 방법은 없는지 기록하세요.

- **성공 요인:** 
	- 이전에는 중간값을 구하려고 복잡하게 생각했으나, **`(전체 합) - (최댓값)`** 공식을 사용하여 `나머지 두 변의 합`을 간단하게 구해냈다.
	- `Math.max`를 중첩하여 3개의 수 중 최댓값을 한 줄로 찾아내어 코드가 간결해졌다.
- **배운 점:** 
	- 문제에서 요구하는 값(나머지 두 변의 합)을 구할 때, 직접적인 방법(변수 찾기)보다 간접적인 방법(전체에서 빼기)이 훨씬 효율적일 수 있다는 것을 깨달음.
