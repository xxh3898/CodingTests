## 💡 문제 이해
> 문제를 나만의 언어로 요약해서 적어보세요.

- **요약**: 이중 반복문(`for`문)이 주어졌을 때, 코드의 수행 횟수와 시간 복잡도(차수)를 출력하는 문제.
- **핵심 조건**: 
  - 알고리즘 구조:
    ```
    for i = 1 to n - 1
        for j = i + 1 to n
            ...
    ```
  - `i`가 1일 때 `j`는 `n-1`번 반복
  - `i`가 2일 때 `j`는 `n-2`번 반복
  - ...
  - `i`가 `n-1`일 때 `j`는 `1`번 반복
  - 즉, 전체 수행 횟수는 $1 + 2 + \dots + (n-1)$이다.

---

## 🚀 접근 방식 (Logic)
> 코드를 짜기 전, 어떤 알고리즘과 자료구조를 쓸지 설계하는 단계입니다.

1.  **수행 횟수 계산 (등차수열의 합)**:
    - $1$부터 $n-1$까지의 합을 구하는 공식은 $\frac{(n-1) \times n}{2}$이다.
    - 예: $n=3$이면 $(2 \times 3) / 2 = 3$번 수행.
    - 예: $n=4$이면 $(3 \times 4) / 2 = 6$번 수행.
    
2.  **자료형 선택**:
    - $n$의 최댓값은 $500,000$.
    - $n^2$은 약 $2,500$억이므로 `int` 범위를 넘는다.
    - 따라서 **`long`** 타입을 사용해야 한다. (이미 잘 쓰셨습니다!)

3.  **최고차항 차수**:
    - 수행 횟수 $\frac{n^2 - n}{2}$에서 최고차항은 $n^2$이므로 차수는 `2`이다.

## 💻 소스 코드 및 문제 설명
> 백준허브가 자동 생성한 **문제 폴더**로 이동합니다.

- [📂 문제 풀이 폴더 열기 (소스코드 & README)](../../백준/Bronze/24265. 알고리즘 수업 － 알고리즘의 수행 시간 4/)

```java
import java.util.Scanner;

public class Main {

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        long n = sc.nextLong();

        // 등차수열 합 공식: n * (n - 1) / 2
        // 주의: (n * n - 1) / 2 가 아님!
        System.out.println(n * (n - 1) / 2);
        System.out.println(2);
    }
}
```

## 📝 오답 노트 & 회고
> 틀렸다면 왜 틀렸는지, 맞았다면 더 좋은 방법은 없는지 기록하세요.

- **틀린 이유 (수식 오류):** 
	- 등차수열의 합 공식 $\frac{n(n-1)}{2}$을 `(n * n - 1) / 2`로 잘못 작성함.
	- 전개해보면 정답은 $\frac{n^2 - n}{2}$인데, 작성한 코드는 $\frac{n^2 - 1}{2}$이라 결과가 다르게 나옴.
- **배운 점:** 
	- $1$부터 $N$까지의 합 공식을 코드로 옮길 때 괄호 위치와 연산 순서를 한 번 더 확인해야 함.
	- 테스트 케이스(예: $n=3, n=4$)를 직접 대입해서 내 공식이 맞는지 검증해보는 습관이 필요함.
