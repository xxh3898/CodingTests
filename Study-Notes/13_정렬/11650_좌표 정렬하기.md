# 2026-03-19
> N개의 평면 위 점을 (x, y) 순으로 정렬하는 문제입니다.

- **핵심 조건**: 
    1. x좌표가 증가하는 순으로 정렬한다.
    2. x좌표가 같으면 y좌표가 증가하는 순으로 정렬한다.

---

## 🚀 접근 방식 (Logic)
> 2차원 배열과 정적 메서드인 Arrays.sort()를 활용하여 정렬 기준을 정의합니다.

1. **변수 설정:** `int[][] arr = new int[n][2]` (n개의 좌표, 각 좌표당 x, y 저장)
2. **조건문 설계 (Comparator):** 
    - `e1[0]`와 `e2[0]`이 같다면 `e1[1] - e2[1]` (y기준 오름차순)
    - 다르면 `e1[0] - e2[0]` (x기준 오름차순)
3. **시간 복잡도:** $O(N \log N)$ - 자바의 Dual-Pivot Quicksort 또는 TimSort 사용

## 📝 오답 노트 & 회고
> 처음 시도 시 2차원 배열 문법과 정렬 방식에서 시행착오가 있었습니다.

- **틀린 이유:** 
    - `Arrays.sort(arr)` 만으로는 2차원 배열의 정렬 기준을 알 수 없어 런타임 에러 발생 가능성.
    - `bw.write()`에 인자를 넣지 않거나, `br.readLine()`으로 공백 데이터를 한꺼번에 처리하지 못함.
- **배운 점:** 
    - **2차원 배열 정렬**: `Arrays.sort(arr, (e1, e2) -> { ... })` 람다식을 통해 사용자 정의 정렬이 가능하다는 것을 학습.
    - **입력 처리**: `StringTokenizer`를 사용하여 한 줄에 공백으로 구분된 데이터를 효율적으로 파싱하는 방법 복습.
- **다른 사람 풀이 참고:** Comparator를 따로 익명 객체로 구현하는 방식보다 람다식이 훨씬 간결함.
