## 💡 문제 이해
> 문제를 나만의 언어로 요약해서 적어보세요.
- 입력으로 **하나의 자연수 N**이 주어진다.
- 이 숫자의 **각 자리 숫자를 내림차순으로 정렬하여 출력**해야 한다.
- 입력은 **공백 없이 하나의 문자열 형태**로 들어온다.

예시

입력
```
2143
```

출력
```
4321
```

- **핵심 조건**
	- 숫자 전체를 정렬하는 것이 아니라 **각 자리 숫자**를 정렬해야 한다.
	- **내림차순 정렬**
	- 입력은 **공백이 없는 문자열**

---

## 🚀 접근 방식 (Logic)
> 코드를 짜기 전, 어떤 알고리즘과 자료구조를 쓸지 설계하는 단계입니다.

### 1. 변수 설정

```
String s → 입력 숫자 문자열
char[] arr → 각 자리 숫자를 저장할 배열
```

또는

```
ArrayList<Integer> arr
```

---

### 2. 조건문 설계

1. 입력 문자열을 **char 배열로 변환**

```
char[] arr = s.toCharArray();
```

2. 배열을 **오름차순 정렬**

```
Arrays.sort(arr);
```

3. 뒤에서부터 출력하여 **내림차순 결과 생성**

```
for (int i = arr.length-1; i >=0; i--)
```

---

### 3. 시간 복잡도

정렬 알고리즘

```
Arrays.sort()
```

Java에서 primitive 배열 정렬은 **Dual-Pivot QuickSort** 사용.

시간 복잡도

```
O(n log n)
```

단, 이 문제의 최대 자리수는 10이므로 실제 수행 시간은 거의 **O(1)** 수준.

## 📝 오답 노트 & 회고
> 틀렸다면 왜 틀렸는지, 맞았다면 더 좋은 방법은 없는지 기록하세요.

### 틀린 이유

초기 코드에서

```
StringTokenizer st = new StringTokenizer(br.readLine());
```

를 사용했다.

`StringTokenizer`는 **공백 기준으로 문자열을 분리**한다.

하지만 문제 입력

```
2143
```

은 공백이 없는 문자열이므로

```
["2143"]
```

하나의 토큰만 생성된다.

결과적으로

```
arr = [2143]
```

이 되어 **정렬 대상이 1개라 정렬 효과가 나타나지 않았다.**

---

### 배운 점

1. **문제 입력 형식을 먼저 확인해야 한다.**
2. **자리수 문제는 문자열 또는 char 배열로 처리하는 것이 일반적이다.**
3. `StringTokenizer`는 **공백 구분 입력에서만 사용하는 것이 안전하다.**
