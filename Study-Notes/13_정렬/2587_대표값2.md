# 2026-03-13
## 💡 문제 이해
> 문제를 나만의 언어로 요약해서 적어보세요.

- **핵심 조건**:  
	- 서로 다른 5개의 자연수가 입력된다.  
	 - 입력된 수들의 **평균값**과 **중앙값**을 출력해야 한다.  
	- 평균은 모든 수의 합을 5로 나눈 값이다.  
	- 중앙값은 정렬했을 때 **가운데 값(3번째 값)**이다.

---

## 🚀 접근 방식 (Logic)
> 코드를 짜기 전, 어떤 알고리즘과 자료구조를 쓸지 설계하는 단계입니다.

1. **변수 설정:**  
	  - `int[] arr = new int[5]` → 입력된 숫자 저장  
	- `int sum = 0` → 평균 계산을 위한 합

2. **조건문 설계:**  
	- 평균 출력: `sum / 5`  
	- 중앙값 출력: 배열 정렬 후 `arr[2]`

3. **정렬 방식:**  
	- 이중 반복문을 사용한 **버블 정렬**

4. **시간 복잡도 계산:**
	 버블 정렬 비교 횟수
	- 바깥 반복문: `n = 5`
	- 안쪽 반복문: `n - 1 = 4`

   총 비교 횟수

   ```
   5 × 4 = 20
   ```

   일반적인 시간 복잡도

   ```
   O(n²)
   ```

   그러나 이 문제는 입력 크기가 **5로 고정**이므로 실제 성능 문제는 발생하지 않는다.
